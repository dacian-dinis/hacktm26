# Wire-service seed photos for Tier 1 pHash matching.
#
# Each line is one image URL. Blank lines and `#` comments are ignored by the
# post-rebase pre-hash script (`apps/api/scripts/build_wire_phashes.py`,
# which will populate `data/wire_phashes.json`).
#
# Why Wikimedia Commons URLs instead of raw Reuters/AP/AFP/EPA URLs?
#   - Wire-service hero photos sit behind paywalls or rotating CDN tokens,
#     which means our pre-hash script and the demo could 404 at any time.
#   - The photos below are the *same images* the wire services distributed
#     during the 2024-25 news cycle (White House / NATO official photographer
#     / Romanian Presidential Administration releases that wire pools
#     redistribute), now hosted on Commons under licenses that allow
#     download for hashing.
#   - The `Special:FilePath/<name>` URL is the documented stable redirect
#     to the current upload URL — the script just follows redirects.
#
# Curated to match the demo narrative (Romania election + NATO + Ukraine).
# Expand toward ~200 entries to widen the wire-match coverage.
